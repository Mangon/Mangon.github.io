    使用Hexo发布个人博客到Github Pages

    2018-08-24
    Hexo, blog, github

    Hexo是一个快速、简洁且高效的博客框架。Hexo 使用 Markdown（或其他渲染引擎）解析文章，在几秒内，即可利用靓丽的主题生成静态网页。

    安装前提

    安装 Hexo 相当简单。然而在安装前，您必须检查电脑中是否已安装下列应用程序：

    Node.js
    Git

    安装Hexo-cli

    在github中创建仓库

    仓库需要以.github.io结尾

    建站

    配置发布参数

    在 _config.yml 中修改参数

    发布

    本地预览

    可以在 http://localhost:5000/ 预览效果

    完成
