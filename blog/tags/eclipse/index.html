    快捷操作

    操作 快捷键
    全选 command A
    复制 command C
    粘贴 command V
    查找 command F
    回退 command Z
    剪切 command X
    退出 command Q
    删除当前行 command D
    放大字体 command +
    缩小字体 command -
    注释/反注释 command /
    打开preference command ,
    代码补全提示 option /
    最大化/还原 control M
    格式化代码 command shift F
    自动导入包 command shift O
