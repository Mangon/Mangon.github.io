    改变placeholder的颜色

    2018-09-05
    css

    我们都知道输入框中改变默认显示的文字是通过改变input的placeholder来实现的

    部分浏览器通过改变类似-webkit-input-placeholder的属性，使得输入框的placeholder也可以有自己的样式。

    大部分情况下修改的都是颜色，不过其它属性也是可以的
