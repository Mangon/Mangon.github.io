    Javascript

    JavaScript一种直译式脚本语言，是一种动态类型、弱类型、基于原型的语言

    TypeScript

    TypeScript 是由微软开源的编程语言。它是 JavaScript 的一个超集，而且本质上向这个语言添加了可选的静态类型和基于类的面向对象编程。
    官网：
    http://www.typescriptlang.org/
    中文学习网站：
    https://www.tslang.cn/
    https://typescript.bootcss.com/

    React

    React 起源于 Facebook 的内部项目，是一个用于构建用户界面的 JavaScript 框架。
    https://www.reactjscn.com/

    Preact

    Preact - 一个只有 3kB 大小的 React 替代品，拥有与 React 相同的 API、组件和虚拟 DOM。
    https://preact.bootcss.com/

    Vue

    Vue.js - 是一套构建用户界面的渐进式框架。
    https://cn.vuejs.org/v2/guide/

    Webpack

    Webpack 是当下最热门的前端资源模块化管理和打包工具。它可以将许多松散的模块按照依赖和规则打包成符合生产环境部署的前端资源。
    https://www.webpackjs.com/concepts/

    Parcel

    Parcel 是一个 web 应用打包工具, 与其他工具的区别在于开发者的使用体验。它利用多核处理器提供了极快的速度, 并且不需要任何配置。
    https://www.parceljs.cn/getting_started.html

    Rollup

    Rollup 是一个 JavaScript 模块打包器，可以将小块代码编译成大块复杂的代码，例如 library 或应用程序。
    http://www.rollupjs.com/

    jQuery

    Query 是一个高效、精简并且功能丰富的 JavaScript 工具库。它提供的 API 易于使用且兼容众多浏览器，这让诸如 HTML 文档遍历和操作、事件处理、动画和 Ajax 操作更加简单。
    https://www.jquery123.com/

    Zepto

    Zepto 是一个轻量级的、针对现代高级浏览器的 JavaScript 工具库，它兼容 jQuery 的 API 。
    https://zeptojs.bootcss.com/
