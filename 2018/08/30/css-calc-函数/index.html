    CSS

    CSS函数之一，可以在CSS中执行计算操作
    使用 calc() 函数计算 <div> 元素的宽度，实现div距离左右边界各50px:

    定义与用法

    calc() 函数用于动态计算长度值，CSS3支持。

    • 需要注意的是，运算符前后都需要保留一个空格，例如：width: calc(100% - 10px)；
    • 任何长度值都可以使用calc()函数进行计算；
    • calc()函数支持 “+”, “-“, “*”, “/“ 运算；
    • calc()函数使用标准的数学运算优先级规则；
