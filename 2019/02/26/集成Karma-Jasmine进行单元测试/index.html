    什么是 Karma

    Karma 是一款JavaScript测试运行容器，它允许Javascript开发者在各种浏览器中运行自己的测试程序。它也可以集成 Istanbul 进行测试覆盖率的统计。通常我们需要结合测试框架及断言库来使用 Karma。

    以下是 Karma 支持的测试框架：

    • Jasmine
    • Mocha
    • QUnit

    更多 Karma 支持的测试框架可以在 该页面 找到。

    以下是 Karma 支持的浏览器：

    • Chrome and Chrome Canary
    • Firefox
    • Safari
    • PhantomJS
    • JSDOM
    • Opera
    • Internet Explorer
    • SauceLabs
    • BrowserStack

    更多 Karma 支持的浏览器可以在 该页面 找到。

    什么是 Jasmine

    Jasmine 是一款知名的 BDD（Beharior-driven Development） Javscript测试框架。它不需要依赖浏览器，DOM或者其它测试框架。因此它可以适配网站、 Nodejs 项目或者任何 Javascript 可以运行的地方。

    运行 Karma 命令

    karma 支持三个命令

    start [<configFile>] [<options>] 启动 Karma 持续执行，也可以执行单次的测试，然后直接收集测试结果
    init [<configFile>] 初始化配置文件
    run [<options>] [ – <clientArgs>] 触发测试运行
